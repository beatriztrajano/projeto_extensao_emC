# projeto_extensao_emC
# Esponja Magic – Sistema de Gestão Comercial

Projeto desenvolvido para a disciplina Projeto de Extensão II do curso de Ciência da Computação da Universidade Veiga de Almeida (UVA).

O sistema tem como objetivo auxiliar a empresa Esponja Magic no gerenciamento de produtos, clientes e vendas, contribuindo para a organização das informações comerciais e para a redução da dependência de plataformas de marketplace.

# Tecnologias utilizadas

* Linguagem C
* HTML
* CSS
* JavaScript
* PHP

# Funcionalidades

* Cadastro de produtos
* Cadastro de clientes
* Registro de vendas
* Controle de estoque
* Geração de relatórios
* Interface web para gerenciamento
